## 💾 Cómo guardar en WhatsApp:

### Android:
1. Abrí el chat de "Tus mensajes fijados" (o creá uno)
2. Pegá cada respuesta
3. Mantené presionado el mensaje
4. Tocá la estrella ⭐ para guardar

### iPhone:
1. Abrí cualquier chat
2. Pegá la respuesta y enviala
3. Mantené presionado el mensaje enviado
4. Tocá "Destacar" ⭐

### WhatsApp Business (recomendado):
1. Andá a Ajustes → Herramientas para empresa
2. Respuestas rápidas
3. Agregar nueva
4. Escribí "/" + palabra clave (ej: /bienvenida)
5. Pegá el mensaje completo
